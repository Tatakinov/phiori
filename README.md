# これは何？

[phiori](https://github.com/phiori/phiori)
を令和の世に復活させる試み。

# 使い方

phiori.dllは同dllがあるフォルダの\*.pyを読み込むようになっています。
したがってファイル構成は

```
ghost/master
    |- phiori.dll
    |- python312.dll
    |- python312.zip
    |- phiori
    |- example.py
```

のようになると思われます。
